## Oblig 2 – Animated Web Poster

**Title:** Webben og planeten vår  
**Description:**  
This web poster is a digital re-creation of NTNU's original print poster about the internet's environmental footprint. It includes custom SVG graphics and animations, organized to follow the visual structure of the printed version.

**Enhancements made:**  
- Full visual re-implementation of the original NTNU poster  
- All icons and SVGs carefully recreated and placed in correct position  
- Text elements placed freely over the design without text boxes, just like the original  
- Background colors matched to the printed layout with smooth transitions  
- Animations added using CSS and `<animateTransform>`  
- Responsive placement improvements  
- Navigation link back to portfolio added
