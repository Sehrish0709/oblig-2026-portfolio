Project 2 – Animated Poster

In this project, I recreated the NTNU poster “Webben og planeten vår” using HTML, SCSS, SVG elements, and CSS animations. The goal was to make the poster look as close to the original design as possible while also adding movement and animations to make it feel more interactive.

I used many separate SVG elements and positioned them with position: absolute, together with top, left, and z-index. This helped me recreate the visual structure and place elements accurately around the poster.

I created different animations using CSS @keyframes. For example, the Wi-Fi icons blink, the clouds and gas elements float, and the sun uses a pulsing effect. I also added rotation animations to some elements to create more movement across the poster.

The animations help guide the user’s attention and make the poster feel more alive without changing the original design too much.

To improve responsiveness, I used clamp(), percentages, scaling, and media queries. Because many elements use absolute positioning, responsiveness was one of the biggest challenges in the project. Even though some elements still need adjustments on very small screens, the layout works well in the main desktop poster view.

I used SCSS to organise the styles and make the code easier to structure and maintain. I also used fluid typography to make text scale better across different screen sizes.

Through this project, I learned more about SVG implementation, CSS animations, responsive design, and how to recreate a detailed visual design using modern CSS techniques.
